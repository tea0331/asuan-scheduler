## 数据限制说明

### 当前问题
1. **数据量不足**: 仅采集到约 200 期，远未达到目标（SSQ 3000+ 期，DLT 2000+ 期）
2. **历史数据缺失**: 数据仅从 2025 年 1 月开始，缺少更早的历史数据
3. **销量和奖池数据缺失**: 原始数据未包含销量和奖池信息

### 问题原因
1. **数据源限制**: 使用的 GitHub 项目仅采集了最近的历史数据
2. **防爬虫策略**: 
   - 500.com 网站启用了 Tencent Cloud EdgeOne 防护，直接访问被拦截（567 错误）
   - 中国体彩网 API 也返回 567 错误（安全防护）
3. **时间限制**: 在当前任务时间内，未能找到完整的历史数据公开 API
