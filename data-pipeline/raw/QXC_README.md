## 数据格式
```json
{
  "lottery": "QXC",
  "issue": "2024150",
  "date": "2026-06-20",
  "numbers": [1, 2, 3, 4, 5, 6, 7],
  "sales": 0,
  "pool": 0
}
```

## 数据说明
- `lottery`: 彩票类型，QXC表示七星彩
- `issue`: 期号，格式为YYYYNNN（年+序号）
- `date`: 开奖日期
- `numbers`: 开奖号码数组，共7位数字
- `sales`: 销售额（元）
- `pool`: 奖池金额（元）
