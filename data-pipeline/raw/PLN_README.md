## 数据格式
```json
{
  "lottery": "PLN",
  "issue": "26001",
  "date": "2026-06-22",
  "numbers": [3, 5, 8, 12, 23, 35],
  "special": 7,
  "sales": 34051363,
  "pool": 135127277
}
```

## 数据说明
- `lottery`: 彩票类型，PLN表示台湾威力彩
- `issue`: 期号，格式为YYNNN（年+序号）
- `date`: 开奖日期
- `numbers`: 开奖号码数组，共6位数字
- `special`: 特别号
- `sales`: 销售额（台币）
- `pool`: 奖池金额（台币）
