## 一、本次改动

### 问题

V19上线后发现：同类别新闻（如4条AI新闻）全部输出**完全相同**的影响链和断裂点，无差异。

### 修复

板块一新增**模板去重机制**：

| 场景 | 修复前 | 修复后 |
|------|--------|--------|
| 同类别第1条 | 输出完整5层影响链 | 输出完整5层影响链（含📌触发实体） |
| 同类别第2-4条 | 重复输出完全一样的完整链 | 只标注「实体」触发相同传导链→见上方 |

### 改动文件

| 文件 | 改动 |
|------|------|
| `generate_full_daily.py` | `_format_impact_chain()` 注入实体标签；`_fallback_all_sections()` 板块一新增 `_append_impact()` 去重函数 |

```bash
python3 -c "
from generate_full_daily import _match_impact_chain, _format_impact_chain, _extract_entity
import re

ai_titles = [
    '英伟达发布新一代GPU H200',
    'OpenAI推出GPT-5模型',
    'DeepSeek开源最新大模型',
]
used = set()
for t in ai_titles:
    tpl = _match_impact_chain(t)
    tpl_id = tpl['fracture']
    entity = _extract_entity(t)
    ent_short = re.sub(r'^[在的得了被把将向从]', '', entity[:8]).rstrip('？?！!。、')
    if tpl_id not in used:
        used.add(tpl_id)
        print(f'【首次-完整链】{t}')
        print(_format_impact_chain(tpl, ent_short))
    else:
        print(f'【去重-简标】{t}')
        print(f'  > 📌 同类影响: 「{ent_short}」触发相同传导链')
    print()
"
```

**预期结果**：第1条输出完整影响链，第2-3条只标注「同类影响」。
