## 改动概述

V18及之前的"邪修操作卡"系统（找谁/怎么说/收X%撮合费）**全部删除**，替换为"影响链+资源短缺预警"系统。

**核心理念变化**：不教怎么做生意，只推演哪些资源会短缺/过剩。

---

## 改动清单

| 改动项 | 说明 |
|--------|------|
| 删除 `GRAY_OPS_PLAYBOOK` | 5组灰色操作卡话术（小三通/对敲等） |
| 删除 `_match_gray_ops()` | 灰色操作卡匹配函数 |
| 删除 `_format_xie_xiu_card()` | 5元组操作卡格式化（断裂/找谁/怎么说/资金/撤退） |
| 删除 `_infer_xie_xiu_op()` | 21类话术模板（450行硬编码） |
| 删除 `_infer_signal()` | V14旧版落地动作（死代码） |
| 删除 `_fill_ops_template()` | 操作卡模板填充（死代码） |
| 删除 `_fallback_gap_scan()` | 板块二缺口扫描话术 |
| 新增 `IMPACT_CHAIN_TEMPLATES` | 15类5层影响链模板（稀土/AI/芯片/出口等） |
| 新增 `_match_impact_chain()` | 影响链模板匹配 |
| 新增 `_format_impact_chain()` | 影响链格式化（📡 影响链+⚡ 断裂点） |
| 新增 `_infer_impact_chain()` | 影响链推演（替代 `_infer_xie_xiu_op`） |
| 新增 `_fallback_shortage_alert()` | 板块二资源短缺预警（替代缺口扫描） |
| 新增 `_inject_shortage_alert()` | 板块二注入（替代灰色操作卡注入） |
| 修改 `_fallback_deep_chain()` | 14处"邪修之道: 找谁→收X%"改为"商机定位: 断裂层+短缺+窗口" |
| 修改 AI prompt | 板块一"落地动作"→"影响链"；板块二"缺口扫描"→"资源短缺预警"；板块四"收过路费"→"商机定位" |
